Top 5 HTML tags:

IMAGE TAG:
The image tag is very useful because it allows you to add pictures to your website, you just need to follow the patch from where it came from.
HEADING TAG:
This tag helps break up text and make the heading and titles stand out from standard text with different levels of font size, h1, h2, h3...
LINK TAG:
This tag helps connect websites from the web by including links or to contect more of your html pages in the navigation area.
PARAGRAPH TAG:
This tag helps seperate context within the same heaind by grouping it together and providing space inbetween.
HTML / HEAD / BODY TAGS:
These tags are very important to keep the proper structure of an html file and help others if they are reading your code.
